# arkui-static-xts-generator（须自行下载，不随本 skill 内嵌）

本目录**不包含** `categories/`、`common/`、`SKILL.md` 等正文，避免与上游重复维护。使用 ohxtsstatic 前请按下述方式获取资料并放到**本路径**（与 `ohxtsstatic/SKILL.md` 同级下的 `arkui-static-xts-generator/`）。

## 下载来源

开源仓库：**[arkUISkill（GitCode）](https://gitcode.com/qq_44921954/arkUISkill)**

## 放置步骤

1. 克隆或下载该仓库。  
2. 将仓库内的 **`arkui-static-xts-generator`** 文件夹**整体拷贝**到本目录，使本地存在：  
   - `arkui-static-xts-generator/categories/`  
   - `arkui-static-xts-generator/common/`  
   - `arkui-static-xts-generator/SKILL.md`（生成器原文，可选阅读）  
3. 拷贝完成后，**不要**删掉本 `README.md`（或可与上游合并时保留其中「下载说明」一节）。

## 与 IDE（双份 skill）路径

若使用 **`.claude/skills/ohxtsstatic/`** 下的同名 ohxtsstatic（编辑器侧技能目录），请在 **`ohxtsstatic/arkui-static-xts-generator/`** 下同样放置一份（或符号链接到同一份目录）。

历史上曾使用 **`arkui_static_xts`**；与上游 **`arkui-static-xts-generator`** 对齐后，以本目录名为准。
